## ✅ Tasks Completed

### Task 1 — Term Deposit Prediction
**File:** `term_deposit_prediction.ipynb`

Built a binary classification model to predict whether a bank customer will subscribe to a term deposit, based on demographic and campaign data.

**What it achieved:**
- Cleaned and preprocessed real-world banking data
- Applied machine learning classification algorithms
- Evaluated model performance using accuracy, precision, and recall metrics
- Demonstrated ability to handle imbalanced datasets and categorical features

---

### Task 2 — Customer Segmentation
**File:** `Customer_Segmentation_Complete.ipynb`

Performed unsupervised learning to group customers into meaningful segments based on purchasing behaviour.

**What it achieved:**
- Applied clustering techniques (K-Means) to identify distinct customer groups
- Visualised customer segments to derive actionable business insights
- Helped understand customer personas for targeted marketing strategies

---

### Task 3 — Global Superstore Interactive Dashboard
**Files:** `app.py`, `Global_Superstore.csv`

Built a fully interactive business intelligence dashboard using Streamlit and Plotly, based on the Global Superstore dataset from Kaggle.

**What it achieved:**
- Displayed key KPIs: Total Sales, Total Profit, Order Count, and Average Profit Margin
- Enabled dynamic filtering by Region, Category, Sub-Category, and Year
- Visualised sales and profit trends across regions, categories, and time
- Identified top customers by revenue
- Analysed the impact of discounts on profitability via scatter plots
- Deployed as an interactive web app accessible without any coding knowledge
